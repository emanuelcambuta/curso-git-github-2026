git init .    			    -> Cria um repo git na pasta actual
git status      		  -> MOstra o status do trabalho(de cada arquivo)
git add nome_arquivo 		-> Adiciona arquivo para ser commitado
git commit -m "msg" 		-> Faz commitndo que estava para ser commitado
git log 			-> Mostra todos históricos de commit

Fluxo de trablho

1.criar o diretório
2. iniciar o git dentro do diretório criado(git init .)
3. edita ou criar arquivos
4. git status
5.git add <add nome>
6. git status
7. git commit -m "mensagem estranha "
8. git status
9. volta para o passo 3.
